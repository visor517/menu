## menu

Проект представляет собой REST API для управления меню ресторана. 
Реализована иерархическая структура: меню → подменю → блюда. 
Каждый уровень содержит CRUD операции.
API документировано с помощью Swagger (доступно по адресу localhost:8000/docs).

### Технологии

- **FastAPI** — веб-фреймворк для создания API
- **PostgreSQL** — реляционная база данных
- **SQLAlchemy** — ORM для работы с базой данных
- **pydantic** — валидация данных
- **Docker** — контейнеризация приложения и базы данных
- **Docker Compose** — оркестрация многоконтейнерного приложения
- **Uvicorn** — ASGI-сервер для запуска FastAPI
- **pytest** — фреймворк для тестирования

### Настройка проекта
Создайте `.env` файл в корне репозитория:
```bash
cp .env.dist .env
```

### Сборка, запуск и тестирование
В корне репозитория для сборки и запуска контейнеров выполните команду:
```bash
docker-compose up --build -d
```
Для запуска тестов:
```bash
docker-compose exec backend pytest tests/ -v
```
Для удаления контейнеров с данными БД:
```bash
docker-compose down -v
```
